# 🎬 Watchlist Manager (Python & JavaScript)

## 📌 Overview

This project is a simple **Watchlist Manager** that allows users to:

* Create an account
* Login securely
* Add movies to a personal watchlist
* Remove movies
* View saved movies

The project is implemented in:

* 🐍 Python (CLI-based application)
* 🌐 JavaScript (Browser-based version using prompts)

---

## 🚀 Features

* 🔐 User authentication system (username & password)
* 💾 Data stored using JSON file (Python version)
* 🎬 Add movies to watchlist
* ❌ Remove movies from watchlist
* 📋 Display all movies
* 👤 Separate watchlist for each user

---

## 🛠️ Technologies Used

* Python (JSON, file handling)
* JavaScript (basic browser interaction)
* HTML (for web version)

## ▶️ How to Run

### 🐍 Python Version

1. Make sure Python is installed
2. Run the file:

```
python watchlist.py
```

---

### 🌐 JavaScript Version

1. Open `index.html` in any browser
2. Use popup prompts to interact
